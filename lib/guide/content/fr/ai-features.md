# Fonctionnalités IA

Move to AI intègre plusieurs fonctionnalités d'intelligence artificielle directement dans votre workflow.

## Copilot IA

Le Copilot est votre assistant conversationnel qui connaît l'ensemble de vos données de transformation.

**Exemples de questions :**
- "Quelles sont les 3 opportunités avec le meilleur ROI estimé ?"
- "Résume l'avancement de notre transformation IA ce trimestre"
- "Quels processus n'ont pas encore d'opportunité IA identifiée ?"

## Briefing hebdomadaire

Chaque semaine, l'IA génère automatiquement un briefing synthétisant :
- L'avancement des initiatives clés
- Les nouveaux cas d'usage créés
- Les alertes et risques identifiés
- Les recommandations prioritaires

Accessible dans **Insights → Briefing**.

## Scenarios IA

Le moteur de scénarios vous permet de comparer différentes approches pour un projet IA :
- Définissez 2 à 4 approches alternatives
- L'IA analyse les trade-offs (coût, délai, risque, ROI)
- Obtenez une recommandation argumentée

## Tech Radar

Le radar technologique classe automatiquement vos technologies en 4 anneaux :
- **ADOPT** : En production, recommandé
- **TRIAL** : En expérimentation
- **HOLD** : À éviter ou à remplacer
- **EMERGING** : À surveiller
