# Gérer les opportunités IA

Les opportunités représentent le **pipeline** de votre transformation IA.

## Créer une opportunité

1. Cliquez sur **"Nouvelle opportunité"** depuis le tableau de bord ou la vue pipeline
2. Renseignez le titre, la description du problème et le processus concerné
3. Estimez la valeur potentielle et le niveau d'effort
4. Assignez un responsable

## Cycle de vie d'une opportunité

| Étape | Description |
|-------|-------------|
| Identifiée | L'idée a été captée |
| Qualifiée | La faisabilité a été vérifiée |
| En cours | L'implémentation a démarré |
| Déployée | La solution est en production |

## Bonnes pratiques

- Soyez précis dans la description du **problème** (pas de la solution)
- Estimez un ROI réaliste avec des hypothèses documentées
- Associez l'opportunité au **processus métier** concerné pour faciliter l'analyse

## Kanban des opportunités

La vue kanban permet de visualiser et de faire glisser les opportunités d'une étape à l'autre. Chaque colonne correspond à une étape du cycle de vie.
